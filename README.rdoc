Reto CRUD TODO list

=== ENDPOINTS

URL: /todos, VERBO: GET  
Retorna la lista de todos.

URL: /todos, VERBO: POST 
Crea un todo y retorna el todo creado.

URL: /todos/:id, VERBO: GET 
Retorna un todo.

URL: /todos/:id, VERBO: PUT
Actualiza un todo.

URL: /todos/:id, VERBO: DELETE 
Elimina un todo y devuelve el todo eliminado.
